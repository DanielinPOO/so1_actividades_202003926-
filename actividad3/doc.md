## 1. Creacion de usuarios
**Para crear tres usuarios llamados `usuario1`, `usuario2` y `usuario3`, puedes utilizar el siguiente comando en la terminal:**
**Comando:**
```bash
sudo useradd usuario1
sudo useradd usuario2
sudo useradd usuario3
